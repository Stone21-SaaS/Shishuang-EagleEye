# 贡献指南

欢迎参与石霜鹰眼的开发！请遵循以下规范：

---
## 📌 贡献流程
1. Fork 仓库
2. 创建 Feature 分支 (`git checkout -b feature/your-feature`)
3. 提交代码 (`git commit -m "feat: add your feature"`)
4. 推送到分支 (`git push origin feature/your-feature`)
5. 创建 Pull Request

---
## 📜 代码规范
- 遵循 PEP 8 规范
- 使用类型提示（Type Hints）
- 添加必要的单元测试

---
## 🚀 Pull Request 规范
- 标题清晰，描述详细
- 每个 PR 只解决一个问题
- 确保所有测试通过
